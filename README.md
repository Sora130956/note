# 编程学习博客文章整理指南

## 目录结构

```
note/
├── backend/
│   ├── java/         # Java相关博客笔记
│   ├── python/       # Python相关博客笔记
│   └── nodejs/       # Node.js相关博客笔记
└── templates/        # 模板文件
    └── template.md   # Markdown笔记模板
```

## 操作流程

1. **提供博客链接**：向我发送你想要整理的编程学习博客文章链接
2. **内容获取**：我会使用WebFetch工具获取网页内容
3. **内容整理**：根据模板将内容整理成Markdown格式
4. **保存文件**：将整理好的内容保存到对应技术类别的目录中
5. **修改优化**：根据你的反馈进行修改和优化

## 模板结构说明

Markdown笔记模板包含以下部分：

- **元数据**：来源、作者、发布日期、分类、标签
- **目录**：自动生成的内容目录
- **摘要**：文章核心内容的简要总结
- **主要内容**：按章节组织的文章主体
- **代码示例**：文章中的代码示例，带有语法高亮
- **关键点总结**：提取的重要知识点
- **实践建议**：从文章中获取的实践建议
- **参考资料**：相关的参考链接

## 如何使用

1. 发送博客文章链接给我
2. 我会根据链接获取内容并整理成Markdown格式
3. 保存到对应的技术类别目录中
4. 你可以查看生成的Markdown文件并提出修改意见
5. 我会根据你的意见进行修改和优化

## 示例

例如，如果你发送一个关于Java并发编程的博客链接，我会：

1. 获取网页内容
2. 使用模板整理成Markdown格式
3. 保存到 `backend/java/` 目录中
4. 提供文件路径给你查看
